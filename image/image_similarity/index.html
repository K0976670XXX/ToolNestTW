相似圖片整理

純前端運算，在您的瀏覽器中尋找並分類相似的圖片，支援輸出分類腳本或打包下載。

輸入

拖曳圖片或資料夾到這裡（支援多重檔案）

或手動選擇多張圖片。所有處理都在本地進行，保護您的隱私。

操作設定

目前容差: 0.3 (值越小要求越嚴格)

掃描進度：0 / 0

輸出結果

掃描結果將會顯示在這裡。

使用方式

1. 上傳或拖曳多張圖片進入虛線框。

2. 調整「相似度容差值」，預設 0.3 能找出大部分視覺相近的圖片。

3. 點擊「開始掃描」進行特徵交叉比對。

4. 掃描完成後，您可以預覽相似群組，並選擇下載 .bat 腳本（用於在本地快速移動或命名檔案），或直接下載分類好的 .zip 包。

.bat 腳本使用教學

下載 .bat 腳本後，請將該檔案移動到您原始圖片所在的資料夾中，然後點擊執行即可。以下為兩種模式的執行示範：

常見問題

圖片會上傳到伺服器嗎？ 不會。所有處理都會在您的瀏覽器中本地執行，過程既快速又安全。

.bat 腳本怎麼用？ 下載後將它與原始圖片放在同一個資料夾，點擊執行即可自動為您分類/重新命名圖片檔案。
